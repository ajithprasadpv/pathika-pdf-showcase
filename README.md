## Overview

Pathika is a lightweight, privacy-focused PDF viewer built for Android. No ads, no tracking, no internet permission required — just a clean and fast reading experience.

---

## Key Features

### PDF Viewing
- Smooth vertical scrolling with momentum
- Pinch-to-zoom and double-tap to zoom
- Page rotation (90°, 180°, 270°)
- Night mode with adjustable brightness
- Page fit modes: Fit Width, Fit Height, Fit Page

### Navigation
- Bottom thumbnail filmstrip with page previews
- Tap-to-jump page indicator
- Jump to specific page dialog
- Auto-hide controls after 4 seconds of inactivity

### Home Screen
- Recent files list with last-read position memory
- Favorites tab for bookmarked PDFs
- Material Design 3 with dynamic colors

### Settings
- Theme: Light, Dark, or System Default
- Default fit mode preference
- Thumbnail position (Bottom or Side)
- Night mode brightness adjustment
- Clear history option

### Privacy & Performance
- No ads, no tracking, no data collection
- No internet permission required
- Efficient memory management with bitmap caching
- Hardware-accelerated rendering

## Tech Stack

| Layer | Technology |
|---|---|
| Language | Kotlin |
| UI | Jetpack Compose + Material 3 |
| Architecture | MVVM + Clean Architecture |
| PDF Rendering | Android PdfRenderer API |
| Local DB | Room (bookmarks/history) |
| Preferences | DataStore |
| Image Loading | Coil (thumbnails) |
| Navigation | Jetpack Navigation Compose |

## Requirements

- **Minimum SDK:** Android 7.0 (API 24)
- **Target SDK:** Android 14 (API 34)
- Supports phones and tablets
